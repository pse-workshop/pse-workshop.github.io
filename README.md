# pse.github.io

Website for **Perspectives on Scientific Error** — an annual workshop series on statistical, philosophical, and meta-scientific issues of scientific error.

This is the site for **PSE9 (2027)**, the 9th edition of the workshop.

Built with [Jekyll](https://jekyllrb.com/) and hosted on GitHub Pages. All content is written in Markdown.

## Pages

- `index.md` — Home / overview
- `program.md` — Workshop program
- `call-for-abstracts.md` — Abstract submission
- `organizers.md` — Organizing committee
- `previous-editions.md` — History of the workshop series

## Local development

```bash
bundle install
bundle exec jekyll serve
```

Then open `http://localhost:4000` in your browser.
